📲 Install on Android
Chrome only shows "Add to Home screen" for websites (https://) — not local files. Choose one free method below:
✅ App is already installed on this device!
⭐ Method 1 — Netlify Drop (Easiest, 2 min)
1
On any device open Chrome and visit:
app.netlify.com/drop
2
Upload the hangman_game.html file — Netlify gives you a free https:// link
3
Open that link in Chrome on this phone → tap ⋮ menu → "Add to Home screen" ✅
✅ Free forever. Share the same link with all your customers on WhatsApp!
Method 2 — GitHub Pages (Free, Permanent)
1
Go to github.com → create free account → New repository named hangman (Public)
2
Upload hangman_game.html, rename it index.html
3
Settings → Pages → Source: main → Save. Your link: yourusername.github.io/hangman
4
Open that link in Chrome → ⋮ menu → "Add to Home screen" ✅
Method 3 — Local WiFi (No Internet Needed)
1
Install "Simple HTTP Server" from Play Store on one phone
2
Start server, point to the file. Note the IP shown e.g. 192.168.1.5:8080
3
On customer's phone (same WiFi) open Chrome, type that IP → ⋮ → "Add to Home screen" ✅
⚠️ Both phones must be on the same WiFi network
